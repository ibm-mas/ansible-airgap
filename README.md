# Ansible Airgap Collection for IBM Maximo Application Suite

## Installation
Install the collection direct from [Ansible Galaxy](https://galaxy.ansible.com/ibm/mas_airgap)

```
ansible-galaxy collection install ibm.mas_airgap
```
